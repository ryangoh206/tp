## Overview
**PowerRoster** is a **desktop client management application** for **freelance personal fitness trainers**.

It helps trainers **manage their client roster** by keeping important information such as client details, notes, locations, workout plans, session rates, body measurements, and workout session logs. in a **single centralized application**, optimised for text commands.

## Main Features
PowerRoster currently supports:
- Managing clients (`add`, `edit`, `delete`, `list`, `view`)
- Searching and organizing (`find`, `filter`, `sort`)
- Trainer-specific updates (`note`, `plan`, `rate`, `measure`, `status`)
- Workout tracking (`log`, `last`)

Note: this project is still under development and the features and documentation will be improved along the way.
